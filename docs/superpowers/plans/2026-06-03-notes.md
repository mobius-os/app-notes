# Notes for Möbius — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: use superpowers:subagent-driven-development
> (recommended) or superpowers:executing-plans to implement this plan task-by-task.
> Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a beautiful, offline-capable, git-tracked Möbius notes app —
markdown that renders live as you type (CodeMirror 6 + KaTeX), inline images &
files, pinning, color, search — whose offline edits 3-way-merge on reconnect and
fall back to an agent on true conflicts, and whose notes the dreaming agent can
read as plain markdown.

**Architecture:** Two repos. (A) A small **mobius platform PR** vendors CM6 +
KaTeX under `/vendor` (like three/React). (B) The **`mobius-os/app-notes`** app:
a single `index.jsx` plus cron jobs. Notes are plain markdown files
(frontmatter + body); attachments are content-addressed blobs; offline edits
land on **per-device draft paths** (never the canonical path, so the
last-write-wins outbox can't clobber a merge); reconcile does fast-forward /
diff3; true conflicts become **immutable descriptors** resolved by a **leased,
verify-before-write agent** (cron-CLI, or "Resolve now" via a new chat); a cron
**git snapshot** keeps history.

**Tech stack:** React 19 (importmap), CodeMirror 6 (vendored, minimal modules),
KaTeX (vendored), marked + DOMPurify (esm.sh, card previews), node-diff3
(esm.sh) for 3-way merge, `window.mobius.storage`, OS cron + `claude -p`, git.

**Conventions (load-bearing):**
- Mobius edits happen in a **worktree**, never the main checkout
  (`scripts/mobius-session.sh notes-vendor`). Bind-mounts resolve under the
  worktree. Live tests target **mobius-test (port 8001)**, never prod.
- App pure-logic modules live in `src/lib/*.js` and are unit-tested with
  `node --test`; `index.jsx` imports them. The shipped artifact is still one
  esbuild-compiled module, but the logic is testable in isolation.
- Commit after each green step. Never claim done without running the command and
  seeing the output.
